# testing/httpi

This directory is reserved for the fixtures, flows, and judge-oriented validation assets used to test `httpi` itself and to support LLM-as-a-judge workflows.

## Planned layout

```text
fixtures/  sample payloads, env files, definitions, golden artifacts
flows/     canonical end-to-end request and run scenarios
judge/     pass/fail checklists and validation guidance for coding agents
```
